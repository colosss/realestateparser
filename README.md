# Аналитический парсер Avito и Cian

Этот проект предназначен для сбора, анализа и визуализации данных с платформ Avito и Cian. Он включает в себя скрипты для парсинга данных, выполнения аналитических расчетов и построения графиков для лучшего понимания рыночных тенденций.

## О проекте

Проект автоматизирует процесс извлечения данных с популярных российских онлайн-платформ объявлений Avito и Cian. Собранные данные затем обрабатываются и анализируются для выявления закономерностей, трендов и инсайтов, которые визуализируются в виде понятных графиков.

## Основные возможности

*   **Парсинг данных:** Эффективный сбор информации о объявлениях с Avito и Cian.
*   **Анализ данных:** Глубокий анализ собранных данных с использованием библиотек Python, таких как Pandas и NumPy.
*   **Визуализация:** Создание информативных графиков и диаграмм для наглядного представления результатов анализа.
*   **Централизованное управление:** Единая точка входа (`main.py`) для запуска всего пайплайна.

## Источники данных

*   Avito (Недвижимость, Автомобили и т.д.)
*   Cian (Недвижимость)

## Технологии

*   **Язык:** Python 3.x
*   **Управление зависимостями:** Poetry
*   **Анализ данных:** Pandas, NumPy
*   **Визуализация:** Matplotlib, Seaborn
*   **Парсинг:** BeautifulSoup, Scrapy (предположительно, или стандартные библиотеки Python)

## Установка и запуск

### Установка зависимостей

1.  Клонируйте репозиторий:
    ```bash
    git clone <URL_репозитория>
    cd parser
    ```
2.  Установите зависимости с помощью Poetry:
    ```bash
    poetry install
    ```

### Запуск проекта

Для запуска основного скрипта анализа, который выполняет парсинг, обработку и визуализацию данных, используйте команду:

```bash
poetry run python main.py
```

Отдельные скрипты также могут быть запущены напрямую:
*   Парсинг Avito: `poetry run python avito_parser.py`
*   Парсинг Cian: `poetry run python cian_parser.py`
*   Аналитический скрипт: `poetry run python analitic_script.py`

## Аналитические графики

Результаты анализа визуализируются и сохраняются в директории `analysis_plots/`. Доступные графики включают:

*   `boxplot_price_by_time.png`: ящик с усами цены в зависимости от времени.
*   `boxplot_price_by_type.png`: ящик с усами цены в зависимости от типа (например, квартиры, дома).
*   `correlation_matrix.png`: матрица корреляций между различными признаками.
*   `histogram_price.png`: гистограмма распределения цен.
*   `scatter_price_vs_space.png`: диаграмма рассеяния цены относительно площади.
